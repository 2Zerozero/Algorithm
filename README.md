# Algorithm

## 알고리즘 유형 공부

### 기초

- 완전탐색 (For반복문)
- 최적화 (정수론)
- 기억 (누적합)

### 심화

- 완전탐색 (재귀, 백트래킹)
- 최적화 (재귀, 백트래킹의 경우의 수)
- 기억 (탐다운 DP, 메모이제이션)
- 활용 (바텀업 DP)

### 추가 공부

- 2차원 DP
- LIS / LCS
- 투포인터
- 이분탐색
- DFS / BFS
- 다익스트라
- 트리
- 유니온파인드
- 최소 스패닝 트리
